# Clone Repository

If you want to start developing on a repository (see [create app](create-app.md) on how to create one), you need to locally clone that repository and then open the workspace. Please note that installing **Git** is a prerequisite for this to work. You can find the installer [here](https://git-scm.com/install/).

[!INCLUDE [Clone Repository Steps](../includes/clone-repository-steps.md)]

With that, you are ready to start developing. You might want to [create a container](create-container.md) and [set up your launch.json file](../shared/create-launch-json.md) to start publishing and debugging

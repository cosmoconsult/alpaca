# Free Trial

You can try out COSMO Alpaca for free and run containers for up to 50 hours. There is no prior registration needed. The only thing you need to do is to install the [COSMO Alpaca VS Code extension](../getting-started/setup-vsce.md).

## What You Can Do in the 50-Hour Free Trial

The 50-hour free trial gives you full access to COSMO Alpaca — all features are available without any restrictions, so you can experience firsthand how it streamlines your Business Central projects. Here's what you can explore:

### Set Up a Project in Minutes
Create a fully configured project — including a code repository, automated build and deployment pipelines, and all necessary infrastructure — with just a few clicks. What typically takes hours of manual setup is done for you automatically.

### Spin Up Business Central Environments on Demand
Launch ready-to-use Business Central environments in the cloud without any local hardware or technical setup. Whether you need an environment for development, testing, or a quick demo, simply choose your Business Central version and country, and COSMO Alpaca takes care of the rest.

### Experience a Modern Development Workflow
Follow a complete end-to-end workflow: create a task, develop a change, have it automatically built and tested, review it, and deploy it — all from a single interface. See how automated quality checks catch issues early and how deployments can be managed across different stages like test, QA, and production.

### Collaborate Across Roles
Invite team members — developers, consultants, and project managers alike — to collaborate on the same project. COSMO Alpaca adapts its interface to each user's role, so even non-technical team members can create environments, review progress, and verify changes without any DevOps expertise.

### Manage Costs Transparently
Experience the usage-based pricing model with full transparency. Start and stop environments on demand, set up automatic schedules to avoid unnecessary costs, and get a clear overview of your usage at any time. Stopped environments are free — you only pay for what you actually use.

### Try It in the Browser
Access COSMO Alpaca directly in your web browser at [vscode.dev](https://vscode.dev) — no installation required. This is ideal for consultants or decision-makers who want a quick look without setting up a local development environment.

> [!NOTE]
> The free trial includes 50 hours of container runtime. All project management features — such as creating projects, managing tasks, and browsing pipelines — are available without any time limit. We'll contact you when the 50-hour free trial is over.

# Rename Container

If you need to rename a container in Azure DevOps, you can use the "Rename container" action. To rename a container:

1. Right-click on the container you want to rename
2. Select **Rename container**
3. Enter the new name for the container
4. Confirm the rename operation

The container will be renamed and all references to it will be updated. You will see a notification when the rename operation is complete.

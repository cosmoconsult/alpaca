# Delete Container

If you no longer need a container environment, you can delete it using the "Delete container" action. To delete a container, right-click on the container you want to delete and select **Delete container**:

You will be asked to confirm the deletion. Once confirmed, the container will be deleted along with all its associated resources and data.

> [!WARNING]
> Deleting a container is a permanent action. All data in the container will be lost and cannot be recovered. Make sure you have backed up any important data before deleting a container.

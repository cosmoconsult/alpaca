# Open Stuff

When developing Business Central, you might want to quickly access resources in Azure DevOps. The "Open Stuff" command collection allows you to quickly open various resources without having to navigate through the Web UI.

## Open pipeline

You can open a pipeline by right-clicking on a pipeline and selecting **Open pipeline**. The pipeline will open in your web browser.

## Open build

You can open a build run (build log) by right-clicking on a build and selecting **Open build**. The build will open in your web browser.

## Open release

You can open a release by right-clicking on a release and selecting **Open release**. The release will open in your web browser.

## Open pull request

When you have a branch in a pull request, you can right-click on the branch and select **Open pull request**. The pull request will open in your web browser.

# [**Legacy Extension**](#tab/legacy)

1. Open the **Command Palette** (F1).
1. Run the command **COSMO Alpaca (legacy): Clone Container**.
1. Select the container you want to clone from the list of stopped containers.
1. Enter a name for the new container and confirm.
1. Wait until the container is created and shown in the list.
